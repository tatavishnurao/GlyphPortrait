# GlyphForge Roadmap

## Phase 1 (current)
- Gradio + CLI
- Word parsing
- Theme system
- Export presets

## Phase 2
- Better silhouette extraction quality controls
- Optional GPU segmentation hook (Torch)
- Manual mask editing brush in UI

## Phase 3
- Improved layout:
  - multi-angle text
  - line packing
  - face-aware density weighting

## Phase 4
- Poster templates with title/subtitle blocks
- Batch rendering for multiple themes/seeds
- Metadata sidecar (`.json`) next to PNG

## Phase 5
- Performance overlays:
  - render time
  - attempts
  - fill ratio
  - GPU VRAM (if CUDA available)
